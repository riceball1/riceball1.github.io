About Dana Ng

Welcome to my portfolio website. My name is Dana Ng, and I'm based in San Francisco Bay Area. I'm a web developer who loves to build clean, efficent web applications focusing on the full stack. Currently coding in JavaScript and dabbling with Python/Flask, Angular, React/Redux, and Vue.js .
